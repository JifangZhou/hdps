hdps
====

High dimensional propensity score algorithm

Install
-------
This package can be installed using the `devtools` package in R:
```
library(devtools)
install_github("lendle/hdps")
# or if that doesn't work, try
install_git("git@github.com:lendle/hdps")
```

Documentation
-------------
In R:
```
library(hdps)
?hdps
```
